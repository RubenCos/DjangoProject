Solo archivos .csv. Codificación UTF-8.

Formato esperado (cabecera):

titulo, autor, descripcion, genero, anio_publicacion, isbn, disponible

El campo disponible acepta: si / no / true / false / 1 / 0
El campo genero acepta: ficcion, no_ficcion, ciencia, historia, biografia, fantasia, terror, romance, infantil, otro
